docom.vim
========

A comment doc highlighting plugin for C/Cpp, inspired by [rust.vim](https://github.com/rust-lang/rust.vim/).

# Installation

Use [plug](https://github.com/junegunn/vim-plug) to manage dependencies.

```bash
Plug 'condy0919/docom.vim'
```
